## 🛠 Tools & Methodology

- **Format**: All diagrams are written using **Mermaid.js** syntax embedded in Markdown.
- **Accuracy**: Based on direct analysis of the project's React Native (Expo) source code, ViroReact integration, and Spring Boot/Python backend communication patterns.
- **Usage**: These documents are intended for architectural review, onboarding, and presentation purposes.

> [!TIP]
> To view these diagrams visually, use a Markdown editor with Mermaid support (like VS Code with the Mermaid Preview extension) or paste the code into the [Mermaid Live Editor](https://mermaid.live).
